# Versati

Versati is an Android frontend for the [Miniflux](https://miniflux.app) self-hosted feed reader. It is open source and free of ads.

Keep in mind this is a work in progress and a personal project which I work on in my free time. I plan to add more features in the future, depending on my availability.

## Features
At the moment, the following features are supported:
- Light and dark themes
- Basic and API-key authentication
- Searching articles
- Adding, removing and editing categories and feeds
- Refreshing feeds
- Bookmarking articles
- Looking up bookmarks and reading history
- Caching

## Building and installing
Right now, the only way to build the app is to do it yourself using Android Studio—I plan to start releasing pre-built APKs once the project gains enough maturity. Just clone the repository, open it, and generate an APK that can be installed on your device.
